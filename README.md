# Mac Bootstrapping

This repository contains scripts and configurations to bootstrap a new macOS development environment.

## What This Does

This project provides automated setup scripts to configure a fresh macOS installation with:
- Essential command-line tools
- Development environment setup
- Configuration of common applications and preferences
- Installation of useful utilities and applications

## Getting Started

### Prerequisites

- macOS (recommended: macOS 12 or later)
- Internet connection
- Administrative privileges

### Quick Setup

1. Clone this repository:
   ```bash
   git clone <repository-url>
   cd mac-bootstrapping
   ```

2. Make the setup script executable:
   ```bash
   chmod +x setup-opencode-local-ollama-darwin-arm64.sh
   ```

3. Run the setup script:
   ```bash
   ./setup-opencode-local-ollama-darwin-arm64.sh
   ```

## Included Scripts

- `setup-opencode-local-ollama-darwin-arm64.sh`: Main setup script for macOS ARM64 (Apple Silicon) systems
